In order to run the program for parts D,E, and F, simply type the command: python partDEF.python
into the terminal. The interface will prompt you for the pcap file that you would like to read.
After such, it will prompt you for which information you would like to obtain.
To obtain answers for part D, type "1", representing "Find IP Addresses and Service"
To obtain answers for part E, type "2", representing "Calculate Youtube Network Counters"
To obtain answers for part F, type "3", representing "Retrieve Video Segment Time and Size for Youtube"
